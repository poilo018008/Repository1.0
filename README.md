## Описание
Программа формирует новый массив из строк, длина которых меньше либо равна 3 символам из исходного массива строк.

## Примеры работы программы

["Hello", "2", "world", ":-)"]
["2", ":-)"]

"1234", "1567", "-2", "computer science"]
["-2"]

["Russia", "Denmark", "Kazan"]
